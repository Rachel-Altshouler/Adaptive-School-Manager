## 💡 הרעיון

בשנים האחרונות, מצבי חירום לימדו אותנו שהחיים לא עוצרים — והלימודים לא צריכים לעצור אף הם.

כלים קיימים כמו זום נותנים מענה חלקי, אך לא עונים על **מכלול הצרכים של מוסד חינוכי שלם**: ניהול כיתות, מעקב מטלות, הגשות, ציונים ותקשורת רציפה בין מורים לתלמידים.

**Adaptive School Manager** נולד מתוך צורך זה — מערכת אחת, מקיפה ומשולבת, שמאפשרת לבית הספר לתפקד במלואו גם מרחוק.

### הוראות הרצה

**1. Backend**
```bash
cd RachelProject/WebApplication
dotnet restore
dotnet run
```

**2. Frontend**
```bash
cd ריאקט
npm install
npm run dev
```
